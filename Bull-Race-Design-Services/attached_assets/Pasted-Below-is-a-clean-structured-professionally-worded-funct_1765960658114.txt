## **1. Overview**

The **Bull Race Management Application** is a responsive web platform built using **React + Bootstrap** with **Firebase** as backend.
The application manages Bull Race events that are conducted **sequentially category-by-category** and enables:

* Public users to **view race results, ongoing events, upcoming schedules**, and **register bull pairs**.
* Administrators to **configure categories**, **approve registrations**, **conduct races**, track **lap timings**, and **generate leaderboards**.
